## 소셜 로그인 요청 파라미터

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| `provider` | STRING | ✅ | 로그인 제공자 ('google', 'kakao') |
| `provider_subject` | STRING | ✅ | 제공자 고유 ID (구글: sub, 카카오: id) |
| `email` | STRING | | 이메일 주소 |
| `name` | STRING | | 이름 |
| `provider_issuer` | STRING | | 제공자 발급자 (iss) |
