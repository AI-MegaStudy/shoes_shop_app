## 📝 참고사항

1. **기본 API**: 각 테이블의 CRUD 작업 (단일 테이블 중심)
2. **JOIN API**: 화면에서 필요한 복합 조인 쿼리 (여러 테이블 JOIN)
3. **JOIN API는 화면 담당자가 작업** - 화면 요구사항을 더 잘 이해하고 있어야 하므로
4. 화면은 협업이지만 **API 제작은 개별 작업** - 체크리스트가 개별로 만들어짐
