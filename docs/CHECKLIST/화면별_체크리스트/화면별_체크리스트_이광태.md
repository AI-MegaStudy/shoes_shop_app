### 주요 의존 체인

1. **인증 흐름**
   - 사용자 로그인 → 사용자 메뉴 드로어

2. **구매 흐름**
   - 사용자-상품조회 → 상품상세화면 → 상품상세화면-3D 프리뷰 (선택적)

3. **관리자 흐름**

   - 관리자-로그인 → 관리자-개인정보 수정
   - 관리자-주문목록 조회 → 관리자-주문목록 조회(상세화면)
